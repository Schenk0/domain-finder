# Product

## Register

brand

## Users

Anyone brainstorming domain names for projects, apps, or side bets. Creative-leaning people who appreciate good interfaces. They arrive with a project idea, leave with a shortlist of available domains and a direct path to buy.

## Product Purpose

A domain brainstorming tool powered by Porkbun. Enter a project name, get scored candidates across multiple strategies (exact, prefix, suffix, domain hacks, brandable), check availability, star favorites, and buy. It's a fun side project — not production SaaS — so the interface itself is the point. If someone lands on it and doesn't pause to appreciate the craft, it hasn't done its job.

## Brand Personality

Minimal, interesting, novel. Quiet confidence with unexpected details. Not trying to impress with volume — impresses with precision and taste. The kind of tool that makes you want to show someone else.

## Anti-references

- Generic SaaS: no blue-and-white dashboards, no corporate UI patterns, no "startup template" energy
- Overly cute: no cartoon illustrations, no bubbly pastels, no playful-for-its-own-sake
- Dark hacker: no terminal green-on-black, no monospace-everything, no "developer aesthetic"
- Bland minimalism: no sea of gray and white, no stripped-down-to-nothing, no "clean" as a substitute for interesting

## Design Principles

1. **Utility earns beauty.** Every visual choice serves the workflow. Decoration that doesn't help the user find their domain is cut.
2. **Quiet until it matters.** The interface stays calm during browsing, then rewards interaction with considered details — transitions, state changes, micro-feedback.
3. **One surface, total commitment.** Single page, single purpose. No half-measures. Every pixel of this one screen gets full attention.
4. **Familiar mechanics, unfamiliar feel.** Search, filter, table — people know these patterns. The novelty is in how they look and behave, not in reinventing the interaction model.

## Accessibility & Inclusion

WCAG 2.1 AA. Standard contrast ratios, keyboard navigation, screen reader support. Reduced motion alternatives for all animations.
